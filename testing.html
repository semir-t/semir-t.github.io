About me

After PCB layout is completed, few samples of the PCB have to be manufacturer so that the designed can be confirmed. This is done by creating custom test which depend on the components used on the board. If there is a MCU present, then this would mena that simple low level drivers need to be written so that the communication interface between MCU and all the other components can be confirmed.

After PCB layout is confirmed and firmware is realesed first batch of final product can be manufactured. But, for all the device to work as intended, small testing board, called Test Jig Board has to be design which will assure that all of the components of system are working as inteded.

With my experties I can help you develop test procedures for both problems described above.
